# E-Reader

A lightweigt Ebook Reader to work with Mihon.
Files have to be in a folder and in the form of a Image format like png.
The order at which files are displayed is based on alpabetical order.
Image Scaling is implemented.
If you want a way to track your reading progress jsut create a "list.txt" file in your your home folder and then a folder called Dokumente in it where you flace the list file. 
(I might rework this to be more intuitive if i get annoyed with the way it is now)
